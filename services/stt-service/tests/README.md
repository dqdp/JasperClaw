# STT Service Tests

Purpose:
This directory contains service-level contract tests for the internal
transcription API.

Start here:
- `test_transcribe.py`: open when changing the `/transcribe` contract.

Index:
- `conftest.py`: open when changing default STT test configuration.
- `test_health.py`: open when changing liveness expectations.
- `test_transcribe.py`: open when changing transcription request/response rules.
