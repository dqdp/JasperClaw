# Agent API Clients

Purpose:
This directory contains adapters for downstream runtimes and providers used by `agent-api`.

Start here:
- `ollama.py`: open first for model runtime calls and embeddings.

Index:
- `ollama.py`: open when changing model runtime or embedding requests to Ollama.
- `search.py`: open when changing `web-search` provider calls or normalization.
- `spotify.py`: open when changing Spotify adapter behavior or dependency error handling.
