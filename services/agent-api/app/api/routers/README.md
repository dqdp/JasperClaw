# Agent API Routers

Purpose:
This directory holds route modules split by transport surface.

Start here:
- `chat.py`: open first for the main chat completion path.

Index:
- `audio.py`: open when changing voice endpoint stubs or future voice proxy behavior.
- `chat.py`: open when changing chat request handling, streaming, or response adaptation.
- `health.py`: open when changing liveness or readiness endpoints.
- `models.py`: open when changing the public model-listing contract.
