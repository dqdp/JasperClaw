# Agent API API Layer

Purpose:
This directory contains HTTP-layer composition for `agent-api`.

Start here:
- `routes.py`: open first to see which routers are mounted.

Index:
- `deps.py`: open when changing dependency injection for repositories, clients, or settings.
- `routes.py`: open when changing which routers are exposed by the app.
- `routers/`: open when changing route-specific request handling.
